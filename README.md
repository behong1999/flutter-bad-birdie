# 🏸 Bad Birdie - Badminton Training App

A comprehensive badminton training app built with Flutter, designed to help badminton lovers improve their skills through footwork training, tactical analysis, and curated learning resources.

## 📱 Features

### Current (v1.0.0)
- ✅ **Beautiful Material 3 Design** - Modern, clean interface
- ✅ **Multi-language Support** - English and German
- ✅ **Professional UI** - Badminton-themed with smooth animations
- ✅ **Cross-platform** - Built with Flutter

## 🏗️ Development Setup

### Prerequisites
- Flutter SDK 3.38.5+
- Android Studio / VS Code
- Android SDK

### Getting Started
```bash
# Clone the repository
git clone https://github.com/yourusername/bad_birdie.git

# Navigate to project directory
cd bad_birdie

# Get dependencies
flutter pub get

# Generate localization files
flutter gen-l10n

# Run the app
flutter run
```

### Building for Release
```bash
# Build Android App Bundle
flutter build appbundle --release

# Build APK
flutter build apk --release
```

## 🌍 Localization

Currently supported languages:
- 🇺🇸 English
- 🇩🇪 German (Deutsch)

To add more languages, create new `.arb` files in `lib/l10n/`.

## 🏸 For Badminton Players

Bad Birdie is designed by badminton enthusiasts, for badminton enthusiasts. Our goal is to create the ultimate training companion that helps you:
